## 🔐 GOOGLE SIGN-IN SETUP

After deployment, configure Google Sign-In:

1. Go to: https://console.cloud.google.com
2. Select project "7kskillshare"
3. APIs & Services → Credentials
4. Edit your Web OAuth credential
5. Add Authorized JavaScript origin:
   - `https://skillswap-xxxxx.vercel.app` (your domain)
6. Save

---

## 🔗 OPTIONAL: CUSTOM DOMAIN

Want `skillswap.com` instead of `skillswap-xxxxx.vercel.app`?

1. Buy domain (Namecheap, GoDaddy, etc.) ~$10/year
2. In Vercel Dashboard → Settings → Domains
3. Add your domain
4. Update DNS (follow Vercel's instructions)
5. Wait 24 hours for verification ✓

→ Wait a few minutes for changes to take effect

**Issue: Deployment gets stuck**
→ Check internet connection
→ Verify Vercel CLI is installed
→ Try running command again
