⚡ QUICK START (Copy & Paste)
═════════════════════════════════════════════════════════════════

Open PowerShell in your project folder and paste:

────────────────────────────────────────────────────────────────
1. Build web version (takes ~15 minutes)
────────────────────────────────────────────────────────────────

  .\BUILD_WEB.bat

────────────────────────────────────────────────────────────────
2. Install Vercel CLI (takes ~2 minutes)
────────────────────────────────────────────────────────────────

  npm install -g vercel

────────────────────────────────────────────────────────────────
3. Deploy to Vercel! (takes ~2 minutes)
────────────────────────────────────────────────────────────────

  vercel --prod

  Then follow the prompts (just press Enter for defaults)

────────────────────────────────────────────────────────────────

Done! Your app is LIVE on Vercel 🎉

Your URL: https://skillswap-xxxxx.vercel.app

🔐 SETUP GOOGLE SIGN-IN (Optional but Recommended)
═════════════════════════════════════════════════════════════════

After your app is live:

1. Go to Google Cloud Console
2. Find your OAuth credentials
3. Add your new Vercel domain to authorized origins
4. Format: https://skillswap-xxxxx.vercel.app
5. Save
6. Test Google Sign-In on your live app
