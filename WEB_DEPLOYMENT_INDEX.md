## 🛠️ DEPLOYMENT SCRIPTS

**COMMIT_WEB_DEPLOY.bat**
```powershell
.\COMMIT_WEB_DEPLOY.bat
```
- Commits web deployment files
- Pushes to GitHub
- Backs up configuration

**BUILD_WEB.bat**
```powershell
.\BUILD_WEB.bat
```
- Cleans old builds
- Gets dependencies
- Builds web version
- Creates build/web/

**DEPLOY_WEB.bat**
```powershell
.\DEPLOY_WEB.bat
```
- Interactive menu

- Handles everything
